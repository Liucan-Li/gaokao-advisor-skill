# 高考志愿填报顾问

为考生生成结构化的报考方案文档，涵盖各批次录取分数线、学校梯队推荐、专业填报顺序、冲稳保策略。

## 核心原则

1. **数据时效性**：分数线、录取位次、招生计划每年变化，必须搜索当年或最新可用年份的数据，不可仅凭记忆
2. **信息透明**：标注数据来源和参考年份，让考生知道数据的可靠程度
3. **结构化输出**：按照固定模板生成文档，方便考生对比和决策
4. **适配个性化**：支持全国各省份、本科/专科所有批次、各类选科组合

## 工作流程

### Step 1：收集考生参数

如果用户未一次性提供完整信息，逐一询问（每次一个问题）：

| 参数 | 必选/可选 | 说明 |
|------|:--:|------|
| **省份** | 必选 | 高考报名省份，如"河南" |
| **高考年份** | 必选 | 如 2026 |
| **分数** | 必选 | 高考总分或预估分 |
| **选科/文理** | 必选 | 如："物化生"、"物理类"、"历史类"、"文科"、"理科" |
| **性别** | 可选 | 部分专业（如护理、军事）可能有性别要求 |
| **批次定位** | 可选 | 根据分数和当年分数线自动判断属于本科还是专科；若用户提前知道可直接指定 |
| **目标城市** | 可选 | 如"郑州优先"、"只考虑省会"、"省内不限" |
| **专业偏好** | 可选 | 如"工科+财经商贸"、"医学类"、"计算机方向"、"不限" |

### Step 2：搜索当年分数线

搜索以下数据，尽量获取最新年份。**按可靠性从高到低选择数据源**：

**首要目标（最可靠）**：
1. **阳光高考 (chsi.com.cn)**：教育部官方平台，批次线数据最权威。用 agent-browser 打开 chsi.com.cn 或 web_fetch 访问。这是已测试验证成功率最高的数据源
2. **省级教育考试院官网**：搜索"<省份>教育考试院 2026 录取控制分数线"，同样可靠

**备用目标**：
3. **一分一段表**：搜索"<年份>年<省份>高考一分一段表 <选科类型>"，获取考生省内排名
4. **院校投档线**：搜索目标学校的"<学校> <年份>年<省份>录取分数线 <选科> 最低分"

**搜索技巧（已测试验证）**：
- **首选 agent-browser + bing.com**：web_search 对中文高考数据经常返回空结果；agent-browser 打开 bing.com 搜索中文关键词，搜索结果摘要（snippet）中可直接看到分数线数字
- **web_search 作为辅助**：可附带具体数字（如"396分"）帮助定位数据页，但不要作为唯一数据获取手段
- **注意拦截网站**：gaokao.com、gaokao.cn、gk100.com、baijiahao.baidu.com、baike.baidu.com 等域名通常被安全策略拦截，优先使用 chsi.com.cn 和 eol.cn
- **若所有在线搜索均受阻**：明确告知考生当前无法获取实时数据，建议自行访问省级教育考试院官网查看最新批次线；同时使用上一年数据作为参考并标注

### Step 3：定位考生批次 + 选科限制检查

**批次定位**：根据搜索到的批次线，判断考生属于哪个批次：

```
示例：
本科一批: 511分 → 低于
本科二批: 396分 → 低于  
专科批:   185分 → 高于96分 → 目标批次：专科
```

向考生确认批次定位是否准确，然后根据批次筛选学校。

**选科限制检查（重要！）**：根据考生选科和专业偏好，检查是否有选科不兼容的情况：

| 报考专业 | 通常要求的选科 | 风险 |
|----------|:------------:|------|
| 护理学 | 化学 或 生物 | 历史类/纯文科考生**无法报考**护理专业 |
| 临床医学/口腔 | 物理+化学 | 无化学的物理类考生无法报考 |
| 药学/检验 | 化学 | 无化学的考生通常无法报考 |
| 计算机/电子信息 | 物理（部分院校） | 历史类考生选择受限 |
| 学前教育/师范 | 通常无限 | 各选科均可报 |

> **关键规则**：如果考生选科与目标专业不兼容，第一时间告知考生并引导调整专业方向，不要等到生成方案时才提醒。

### Step 4：搜索目标学校

根据考生参数（批次、城市、专业偏好、分数区间），搜索可报考的学校：

1. 搜索"<省份><城市>公办<本科/专科>院校 名单"
2. 搜索每个候选学校的"<学校> <年份>年<省份>录取分数线 <选科> 最低分"
3. 搜索"<学校>专业 学费标准 招生章程"获取专业和费用信息

搜索目标：
- 获取每个候选学校的参考录取分数线区间
- 获取特色专业和优势方向
- 获取学费标准（公办院校通常有统一指导价，可先查省教育厅发布的收费标准）

### Step 5：生成报考方案文档

按以下模板生成完整文档，保存到当前工作目录下，文件名为`<省份>高考<分数>分报考方案.md`。

## 补充说明

- **本科批次**：搜索学费时注意本科和专科标准不同。本科理工科约5,000~8,000元/年，文科约4,000~5,500元/年。四年总费用约8~10万元
- **专科批次**：专科一般为三年制，总费用低于本科。务必给出专升本路径（考试科目+可报考的本科院校和专业）
- **新高考省份**："物理类/历史类"替代传统"理科/文科"，一分一段表按物理/历史分别统计
- **专业优先 vs 学校优先**：分数在批次线边缘→建议保学校（服从调剂）；分数有竞争力→建议保专业
- **搜索受阻时**：如果 web_search、web_fetch、agent-browser 均无法获取完整数据，不要放弃——使用已有数据生成方案，明确标注哪些是推算值，并建议考生联系招生办验证。关键是要给考生一个可操作的方案框架

## 关键提醒

- 所有分数数据必须来自搜索，标注参考年份，不可凭空编造
- 如果搜索不到最新年份的数据（如当年的分数线尚未公布），使用上一年数据并明确标注
- 学费数据优先搜索各校招生章程；若搜索不到，使用省统一指导价并注明
- 文档生成后，提醒考生：建议查阅各校官网最新招生章程、致电招生办确认录取把握、关注省级教育考试院官方公告
