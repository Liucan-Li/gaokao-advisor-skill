# 高考志愿填报顾问 — 使用指南

> 让 Claude 帮你精准定位高考志愿，生成结构化报考方案。

---

## 安装 Skill

### 方法一：项目级安装（推荐）

将 skill 放到你常用的项目下，技能随项目生效：

```bash
# 1. 克隆或下载 skill 文件到你的项目
mkdir -p <你的项目>/.claude/skills/gaokao-advisor/
cp SKILL.md <你的项目>/.claude/skills/gaokao-advisor/

# 2. 在 Claude Code 中打开该项目即可自动加载
cd <你的项目>
claude
```

### 方法二：全局安装（所有项目生效）

将 skill 安装到用户目录，Claude Code 在所有项目中都能使用：

```bash
# 创建用户级 skills 目录（如果不存在）
mkdir -p ~/.claude/skills/gaokao-advisor/

# 复制 skill 文件
cp SKILL.md ~/.claude/skills/gaokao-advisor/
```

### 方法三：通过 Skill 文件安装

如果你拿到了 `.skill` 安装包（`.skill` 扩展名的 zip 文件）：

```bash
# 在 Claude Code 终端中执行
/skill:install /path/to/gaokao-advisor.skill
```

### 验证安装

安装后在 Claude Code 对话中直接说出你的高考志愿需求，skill 会自动触发。你也可以用以下方式验证：

```bash
# 列出已加载的技能
/skills
```

应该能看到 `gaokao-advisor` 在列表中。

### 目录结构说明

安装完成后，目录应如下：

```
<你的项目或~>/.claude/skills/gaokao-advisor/
└── SKILL.md          ← skill 核心文件（必须）
```

> **提示**：项目级 skill 只在当前项目下的对话生效，适合专门用于志愿填报咨询的项目。全局 skill 在所有 Claude Code 会话中都可用，适合随时咨询。

## 快速开始（3 步完成志愿填报）

### 第 1 步：准备好你的基本信息

在对话中直接告诉 Claude 你的情况，例如：

```
我是河南考生，2026年高考，物理类/物化生，考了 420 分。
我想在郑州上学，偏好工科和计算机方向。
```

Claude 会自动触发高考志愿填报顾问 skill，开始为你分析。

### 第 2 步：回答 Claude 的补充问题

如果信息不完整，Claude 会逐一询问（一次一个问题）：

- 性别（部分专业有性别要求）
- 更具体的城市偏好
- 更具体的专业方向

### 第 3 步：获取报考方案文档

Claude 会自动搜索最新分数线数据，生成一份完整的 Markdown 文档，保存在当前目录，文件名为 `<省份>高考<分数>分报考方案.md`。

## 支持的场景

| 场景 | 示例提问 |
|------|---------|
| 本科批次 | "广东物理类 580 分能上什么学校" |
| 专科批次 | "河南理科 300 分能报哪些专科" |
| 文理分科（旧高考） | "四川文科 520 分想学师范" |
| 新高考选科 | "浙江选科物化生 650 分推荐什么专业" |
| 指定城市 | "只考虑武汉的高校，理科 450 分" |
| 指定专业方向 | "湖南物理类 600 分想学临床医学" |

**支持全国所有省份、本科和专科所有批次、新旧高考所有选科组合。**

## 生成文档结构说明

方案文档包含 7 个部分：

| 章节 | 内容 | 对你有什么用 |
|------|------|-------------|
| 一、分数线参考 | 今年各批次分数线 vs 你的分数 | 看清自己的位置 |
| 二、学费参考 | 目标省份公办院校学费标准 | 预算规划 |
| 三、学校梯队速览 | 按分数匹配度排的学校总表 | 快速了解有哪些选择 |
| 四、各校专业详细推荐 | 每个学校推荐什么专业、填报顺序 | 填志愿时直接参考 |
| 五、填报建议顺序汇总 | 按冲稳保策略统一排序 | 一看就懂的填报优先级 |
| 六、志愿填报方案 | 2-3 套完整填报方案 | 不同偏好下的最优解 |
| 七、填报注意事项 | 平行志愿、调剂等提醒 | 避免踩坑 |

## 数据可靠性标注

文档中会标注每项数据的可靠程度：

| 标记 | 含义 |
|:--:|------|
| ✅ 已搜索验证 | 来自 chsi.com.cn（阳光高考）或省级考试院官网 |
| ⚠️ 推算区间 | 基于历年趋势推算，标注推算依据 |
| ❌ 未搜索到 | 数据未能验证，建议自行查阅官方渠道 |

## 使用建议

### ✅ 这样做效果最好

1. **一次性提供完整信息** — 省份、年份、分数、选科一起说，节省对话轮次
2. **提前想好偏好** — 城市偏好、专业方向提前想好，方案更有针对性
3. **分数边缘情况标注** — 如果分数刚好在批次线附近（±10 分），主动说明，Claude 会给出更谨慎的建议

### ⚠️ 需要注意

1. **分数线数据每年更新** — Claude 会搜索最新数据，但如果当年分数线尚未公布，会使用上一年数据并标注
2. **最终确认以官方为准** — 方案是参考性的，填报前务必查阅各校招生章程和考试院公告
3. **学费是估算值** — 实际学费以各校当年招生章程为准

### 🔄 方案不满意怎么办

直接告诉 Claude 你想怎么调整：

- "把计算机方向调高优先级"
- "增加郑州以外的学校"
- "我想看看冲一冲的选择"
- "学费太贵了，换成更便宜的学校"

Claude 会根据你的反馈修改方案。

## 常见问题

### Q: 这个 skill 能查到今年的分数线吗？
A: Claude 会优先搜索阳光高考（chsi.com.cn）和省级教育考试院官网的最新数据。如果当年分数线已经公布，会使用当年数据；如果尚未公布，会使用上一年数据并明确标注。

### Q: 专升本路径是什么？
A: 如果你分数在专科批次，方案文档的第七部分会提供专升本路径说明，包括考试科目和可报考的本科院校。

### Q: 新高考省份支持吗？
A: 完全支持。物理类/历史类、3+1+2、3+3 等各种选科模式都已适配。

### Q: 可以针对某个具体专业询问吗？
A: 可以。比如"物化生 600 分想学口腔医学"，Claude 会重点分析该专业在各校的录取情况。

### Q: 方案文档保存在哪里？
A: 保存在当前对话的工作目录下，文件名为 `<省份>高考<分数>分报考方案.md`。

## 触发条件

只要你的提问涉及"分数 + 学校/专业选择"，skill 就会自动触发。不需要说"请用高考志愿顾问"。

以下关键词都会触发：
- 高考、报考、志愿填报、分数线
- "XXX分能报什么学校"
- "我这个分数能上哪些学校"
- 询问某省份某批次录取情况
- 咨询专科/本科院校选择
